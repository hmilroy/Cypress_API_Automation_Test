## Get the Swagger Petstore Docker image

https://hub.docker.com/r/swaggerapi/petstore

```bash
docker pull bwolmarans/petstore   
```
```bash
docker run -p 8080:8080 bwolmarans/petstore
```

## Access the Swagger Petstore

http://localhost:8080/
